## GettingandCleaningData Project


Using the DataSet from UCI about the Human Activity Recognition Using Smartphones
Description of the DataSet is shown in the link below:
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

In this project, 
training and test sets have been merged into one single data set.
the measurments with mean and standard deviation have been extracted out.
the data set has been labeled with descriptive ativity names.
averages of each variable of each activity and subject have been stored in averaged_data and write into tidy_data.txt.
